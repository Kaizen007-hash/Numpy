# Numpy Examples

This repository contains a Jupyter notebook `Numpy.ipynb` with many small examples demonstrating NumPy (and a few SciPy/Matplotlib/Pandas) operations.

## Contents
- Vector operations: addition, scalar multiplication, dot product, cross product, norm, orthogonality checks
- Matrix operations: addition, scalar multiplication, matrix-vector and matrix-matrix multiplication, transpose
- Linear algebra: determinant, inverse, Cramer's rule, rank/nullity, block matrix multiplication, Kronecker product
- Elementwise operations: Hadamard product, outer product
- Norms: Frobenius norm and matrix norm inequalities
- Traces and trace properties
- Miscellaneous: sparse matrix conversion (SciPy), plotting examples (Matplotlib), simple Pandas DataFrame display, datetime and formatting examples

## Dependencies
- Python 3.8+
- numpy
- scipy (optional — used in a few cells)
- matplotlib (for plotting cells)
- pandas (for the DataFrame example)

Install dependencies with pip:

```
pip install numpy scipy matplotlib pandas
```

## How to use

1. Open the notebook `Numpy.ipynb` in Jupyter Notebook or JupyterLab.
2. Run cells interactively to see printed outputs and plots.

## Notes
- The notebook is a collection of short, standalone examples intended for learning and demonstration. Feel free to extract cells into scripts or expand them into tests.
